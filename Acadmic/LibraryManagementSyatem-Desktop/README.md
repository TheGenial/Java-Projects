# Project Title #
LibraryManagementSyatem-Desktop


# Description #

It is a Desktop based application build by Java GUI (Graphical user Interface)

- User Registration.
- Login: User.
- Logout: User.
- Add Book.
- Search Book (using different parameters).
- Delete Book.
- Edit/ Update Book.
- List Books.

```
* Java
* Java Swing
```
